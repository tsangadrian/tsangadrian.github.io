Code as Art

Digital Meme

Works created for Art-74, these works demonstrate some of the possibilities code can create along the transition into visual media. Code, as a relatively emerging technology in our age, has humbly grown from its binary roots to an accessible tool that anyone can pick up. From computer framework to applications, code now takes on the forefront of new media art as applications such as Processing demonstrate its capabilities. The following works all utilize Processing.

This first work is my shot at creating a self-portrait using the program. Created using a variety of drawn shapes such as ellipses and rectangles, the final form is created through a layering of these coded shapes. During the process, I wanted to try my hand at not relying on the primitive shapes to make a caricature of my face, rather instead I wanted to manipulate overlapping shapes to emulate complex ones.

The second work showcases a game I created, which the objective would be to get the highest score that you can without dropping below a score of zero. While coding, the main visual aesthetic that you see of cascading circles was initially a mistake. However, after contemplating over it, I actually realized it gave the game sort of an odd visual appeal to it. I thought it looked unique, so I decided to keep it. The last piece was my shot at capturing the vaporwave aesthetic. It was created using a homebrewed photoshop tool for Processing, where custom brushes were utilized.
